You can control internationalization files for the website here.

  • Use the "Setup" button, if available, to perform the first-time setup and create the base file.
  • Use the "Refresh Locales" button to sync up the available locales between the CMS and the files on disk.
  • Use the "Extract Messages" button to update PO files with new messages.
  • Use the "Update Translation" button to apply translations from PO files to the contents of the CMS.
